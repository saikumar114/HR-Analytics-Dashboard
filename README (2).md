# Power BI Employee Analytics Dashboard

## Project Synopsis
An Employee Analytics Dashboard created with **Microsoft Power BI** is presented in this project.  
The dashboard offers information on gender diversity, attrition metrics, salary analysis, and workforce distribution.

Key Findings: Total Employees; Active vs. Resigned Employees; Attrition Rate; Department-wise Employee Count; Department-wise Salary Distribution; Gender Distribution; Overview of Employee Status

Tools Used: Excel (Data Preparation) and Microsoft Power BI

Dashboard Features: Department slicer, interactive KPIs
Charts that move
A tidy and expert design

Included Files: Power BI dashboard file `Employee_Analytics_Dashboard.pbix`; dashboard preview image `dashboard.png`

Take note
This dataset was produced for educational and portfolio purposes and is **synthetic (sample data)**.
